Balises HTML

Les balises HTML définissent la structure d'une page. Voici un exemple de balise simple : 

<p>Ceci est un paragraphe en HTML</p>

Attributs HTML

Les attributs fournissent des informations supplémentaires sur les éléments HTML. Par exemple : 

<a href="https://example.com" target="_blank">Lien vers un site externe</a>

Structure du Document

La structure d'un document HTML est essentielle pour créer des pages bien organisées. Un exemple de structure basique : 


<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page</h1>
    <p>Ceci est un paragraphe</p>
  </body>
</html>

Éléments Inline & Block

En HTML, les éléments se comportent différemment en fonction de leur nature (inline ou block). Voici un exemple : 


<div>Ceci est un élément de type block</div>
<span>Ceci est un élément de type inline</span>
